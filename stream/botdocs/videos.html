In the stream, you will see a little TV playing videos. This document will go over how to add videos to the TV.

Getting Started

To request videos, you will need the ID of the video you will request. For example, the ID of https://www.youtube.com/watch?v=dQw4w9WgXcQ is dQw4w9WgXcQ.

There is no limit when it comes to videos except it must be less than 30 minutes. Please note that however, mods can blacklist videos we deem too inappropriate, or that it can get the stream striked.

Media Share Commands

These are the commands for using videos.

Command Description
!vr (VIDEO ID) Adds a video to the queue.
!video Shows the video currently playing.
!skip Vote to skip a video.

That's it.

Yeah, it's really that easy.
