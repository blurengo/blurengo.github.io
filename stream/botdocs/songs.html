Song Requests Commands Document

This document will go over how to use Song Requests.

Getting Started

Using song requests is incredibly easy. Please note that you can only request songs via YouTube chat.

You can request songs from the following OSTs:
(Superstar Saga/Inside Story Remakes are separate numbers from their counterpart games [Journey and Minions])

Name OSTs Extra?
Paper Mario Series 5 No
Mario & Luigi Series 9 No
Super Mario RPG: Legend of The Seven Stars 1 No
Tator Tot Tunes' Paper Mario/Mario & Luigi 8-Bit Remixes 2(?) Yes
Origami King Fan Renditions by Ruven Wegner N/A Yes

Requesting Songs

This is a list of every command you should know about requesting songs.

Command Description
!sr ___ The main deal. This command requests a song.
!songs current Displays the song currently playing.
!songs next Displays the song that is next in Queue.
!songs list Sends the link to the queue.
!songs playlist Sends the link to the playlist.
