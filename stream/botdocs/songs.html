Requesting Songs

This is a list of every command you should know about requesting songs.

Command Description
!sr ___ The main deal. This command requests a song.
!songs current Displays the song currently playing.
!songs next Displays the song that is next in Queue.
!songs list Sends the link to the queue.
!songs playlist Sends the link to the playlist.

How to fix "No results found" or requesting the wrong song

Nightbot is picky when it comes to song queries. If you have issue requesting songs, this may help.

#1: Go to the playlist. (!songs playlist)
#2: Use the search function to look around for the song you want.
#3: Copy the name of the song you want to request.
#4: Paste it under !sr.
